# analyse-ventes-librairie
Data analysis of bookstore sales using Python, statistical tests and customer segmentation to identify business opportunities and purchasing behaviors.

# Analyse des ventes d'une librairie

## Contexte

Ce projet a été réalisé dans le cadre de la formation Data Analyst d'OpenClassrooms.

L'objectif était d'analyser les ventes d'une librairie afin de mieux comprendre les comportements d'achat des clients et d'identifier les facteurs influençant les performances commerciales.

## Objectifs

* Analyser les ventes et le chiffre d'affaires.
* Étudier les comportements d'achat des clients.
* Identifier les segments de clientèle les plus rentables.
* Vérifier des hypothèses métier à l'aide de tests statistiques.

## Outils utilisés

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* SciPy
* Jupyter Notebook

## Compétences développées

* Nettoyage et préparation des données
* Analyse exploratoire des données
* Segmentation client
* Analyse statistique
* Tests d'hypothèses (Pearson, Spearman, ANOVA, Mann-Whitney)
* Data Visualization
* Business Analysis

## Résultats

* Identification des profils clients les plus contributeurs au chiffre d'affaires.
* Analyse des relations entre âge, fréquence d'achat et montant dépensé.
* Mise en évidence de tendances de consommation selon les catégories de produits.
* Recommandations marketing basées sur les données.
